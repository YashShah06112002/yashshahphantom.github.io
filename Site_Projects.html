MY PROJECTS
PROJECT GREEN CORRIDOR
Oct 2021

• This was an individual project taken up by me as a part of learning basics of GIS.
• The project contains a static map of a selected region wherein all the roads were carefully digitized from the available satellite imagery and were assigned an average traffic density.
• This was followed by locating premier hospitals in the region.
• The project was made using ArcMap 10.2 keeping in mind to come out with the quickest path connecting two hospitals and to curb the loss of lives due to vehicular traffic.

TRAVELLING SALESMAN PROBLEM
Mar 22 - May 22

• This was a group project which consisted of four members including myself.
• Our goal was to ask the user to enter the cities he wishes to visit along with the distances between them and then our project would simulate the Travelling Salesman Problem and show the shortest possible path covering all the cities.
• The project was made using Python especially with the help of Tkinter library.
